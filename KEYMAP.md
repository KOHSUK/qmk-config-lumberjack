# Keymap Reference

キーマップ: `kohsuk`
ソース: `qmk/keyboards/peej/lumberjack/keymaps/kohsuk/keymap.c`

---

## レイヤー一覧

| # | 名前 | 起動キー | 用途 |
|---|------|---------|------|
| 0 | BASE | — | デフォルト（QWERTY） |
| 1 | FN1  | `=`キー位置の右スペース ホールド | 数字・括弧 |
| 2 | FN2  | 左スペース ホールド | 記号（Shift系） |
| 3 | FN3  | Enter ホールド | ナビゲーション |
| 4 | FN4  | Backspace ホールド | Shift記号・Ctrlショートカット |
| 5 | FN5  | `MO(FN5)` キー ホールド | 左修飾キー |
| 6 | FN6  | 最右スペース ホールド | 右修飾キー |
| 7 | FN7  | `=` ホールド | ファンクションキー |

タッピング設定（`config.h`）:
- `TAPPING_TERM` = 200ms
- `QUICK_TAP_TERM` = 0（LT キーのダブルタップリピート無効）
- `PERMISSIVE_HOLD` = 無効（スペース連打でのレイヤー誤作動防止）

- `=/FN7` = タップ: `=` / ホールド: FN7 レイヤー

- `LF2`   = タップ: スペース / ホールド: FN2
- `LF1`   = タップ: スペース / ホールド: FN1
- `EF3`   = タップ: Enter / ホールド: FN3
- `BF4`   = タップ: Backspace / ホールド: FN4
- `SF6`   = タップ: スペース / ホールド: FN6

## Layer 1: FN1 — 数字・括弧

右スペース（`LF1`）ホールドで起動。

```
┌───┬───┬───┬───┬───┬───┐ ┌───┬───┬───┬───┬───┬───┐
│   │   │   │   │   │   │ │   │   │   │   │   │   │
├───┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│   │ 1 │ 2 │ 3 │ 4 │ 5 │ │ 6 │ 7 │ 8 │ 9 │ 0 │   │
├───┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│   │   │   │   │   │   │ │   │   │   │   │   │ ` │
├───┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│   │   │ < │ { │ [ │ ( │ │ ) │ ] │ } │ > │ \ │   │
├───┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│   │   │   │   │   │   │ │G⎵ │G= │   │   │   │   │
└───┴───┴───┴───┴───┴───┘ └───┴───┴───┴───┴───┴───┘
```

- `G⎵` = `GUI+Space`（macOS Spotlight / 入力切替）
- `G=` = `GUI+=`（macOS ズームイン）
- Row 3 の括弧配列: `< { [ ( ) ] } > \`（X〜/キー位置）

## Layer 3: FN3 — ナビゲーション

Enter（`EF3`）ホールドで起動。

```
┌──────┬───┬───┬───┬───┬───┐ ┌───┬───┬───┬───┬───┬───┐
│      │   │   │   │   │   │ │   │   │   │   │   │   │
├──────┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│ ESC  │ 1 │ 2 │ 3 │ 4 │ 5 │ │ 6 │ 7 │ 8 │ 9 │ 0 │   │
├──────┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│ G(S5)│   │C← │C↑ │C↓ │C→ │ │ ← │ ↓ │ ↑ │ → │   │   │
├──────┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│      │   │CG←│CG↑│CG↓│CG→│ │   │   │   │   │   │   │
├──────┼───┼───┼───┼───┼───┤ ├───┼───┼───┼───┼───┼───┤
│ CAPS │   │   │   │LG1│LG2│ │   │   │   │   │   │   │
└──────┴───┴───┴───┴───┴───┘ └───┴───┴───┴───┴───┴───┘
```

- `G(S5)` = `GUI+Shift+5`（macOS スクリーンショット）
- `C←` 等 = `Ctrl+Arrow`（単語移動）
- `CG←` 等 = `Ctrl+GUI+Arrow`（macOS Mission Control）
- `LG1/2` = `KC_LNG1/2`（言語入力切替）
